## 📄 Описание страниц

### 1. Главная страница (index.html)
- Hero-секция с градиентом
- Две основные кнопки: "Олимпиадное движение" и "Конкурсное движение"
- О платформе
- Актуальные мероприятия

### 2. О школе (about.html)
- Миссия проекта
- Возможности для участников, педагогов и экспертов
- Статистика достижений

### 3. Мероприятия (events.html)
- Фильтры мероприятий (все/олимпиады/конкурсы/активные/предстоящие)
- Карточки олимпиад с бейджами статуса
- Карточки конкурсов
- Новости и анонсы

### 4. Страница мероприятия (event-detail.html)
- Подробное описание
- Этапы проведения
- Награждение
- Документы для скачивания
- Кнопка регистрации

### 5. Документы (documents.html)
- Общие документы платформы
- Документы олимпиад
- Документы конкурсов
- Возможность скачивания PDF

### 6. Контакты (contacts.html)
- Контактная информация
- Форма обратной связи
- FAQ

### 7. Вход/Регистрация (auth.html)
- Вкладки "Вход" и "Регистрация"
- Выбор типа пользователя (участник/педагог)
- Валидация форм

## 🚀 Как запустить

1. Откройте файл `index.html` в браузере
2. Все страницы связаны через навигацию
3. Для локального сервера (опционально):
   ```bash
   python -m http.server 8000
   # или
   npx http-server
   ```
